# CS-300 Reflection

## What was the problem you were solving in the projects for this course?

The primary problem I was solving throughout this course was designing and implementing software that could efficiently store, organize, and retrieve data. In the final project, I developed an advising assistance program for ABC University that allowed advisors to load course information, print a sorted list of courses, and view course details along with prerequisites. The goal was to create a system that could manage course data accurately while providing fast access to information.

## How did you approach the problem? Consider why data structures are important to understand.

I approached the problem by evaluating different data structures and understanding how each one stores and accesses data. Throughout the course, I worked with vectors, linked lists, hash tables, and binary search trees. Understanding data structures is important because they directly affect the performance and efficiency of a program. Choosing the appropriate data structure allows data to be organized in a way that improves searching, sorting, insertion, and deletion operations.

## How did you overcome any roadblocks you encountered while going through the activities or project?

One of the biggest challenges I faced was translating pseudocode into working C++ code and understanding how pointers and dynamic memory worked within data structures. I overcame these roadblocks by breaking problems into smaller steps, reviewing course materials, testing my code frequently, and carefully analyzing compiler errors. As I continued practicing, I became more comfortable with debugging and implementing the required algorithms.

## How has your work on this project expanded your approach to designing software and developing programs?

This project helped me better understand the importance of planning before writing code. Rather than immediately beginning implementation, I learned to analyze requirements, select appropriate data structures, and consider the efficiency of different solutions. I now place greater emphasis on designing a program's structure before development begins, which results in more organized and effective solutions.

## How has your work on this project evolved the way you write programs that are maintainable, readable, and adaptable?

My work in this course has encouraged me to write cleaner and more organized code. I now use meaningful variable names, comments, modular functions, and consistent formatting to improve readability. I also recognize the value of designing programs that can be modified or expanded without requiring major changes to existing code. These practices make programs easier to maintain, debug, and adapt to future requirements.
